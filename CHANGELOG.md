# Changelog

All notable changes to Orkestra will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-02

### Added
- Initial PyPI release
- CLI tool with commands: new, start, stop, status, list, load, reset, config
- Project template system
- Multi-AI coordination framework
- Democratic voting system
- Safety validation layer
- Self-healing capabilities
- Real-time monitoring
- Comprehensive documentation

### Features
- Support for 5 AI services (Claude, ChatGPT, Gemini, Copilot, Grok)
- Lock-based task coordination
- Audit trail system
- Automatic backup system
- Configuration management
- Rich CLI with colored output

### Documentation
- Installation guide
- Quick start tutorial
- API reference
- Architecture documentation
- Troubleshooting guide
